# move_task_to_back

A plugin that helps you use android moveTaskToBack.

## Getting Started

### get plugin

```yaml
move_task_to_back: ^x.y.z
```

### import
```dart
import 'package:move_task_to_back/move_task_to_back.dart';
```

### use it
```dart
MoveTaskToBack.moveTaskToBack(nonRoot: false/true);
```
